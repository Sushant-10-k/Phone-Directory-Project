## 🚀 Quick Start (Choose One)

### Option 1: Automated Setup (Recommended)

**Windows:**
```bash
setup.bat
```

**Linux/Mac:**
```bash
bash setup.sh
```

### Option 2: Manual Setup

**Terminal 1 - Backend:**
```bash
cd backend
npm install
npm start
```

**Terminal 2 - Frontend (new terminal):**
```bash
cd frontend
npm install
npm start
```

Then open: `http://localhost:3000`

## 📖 Common Tasks

### I want to add a contact
1. Click "Add New Contact"
2. Fill the form
3. Click "Save Contact"

### I want to find a contact
1. Use the search box at top
2. Select search type (Name, Email, Phone)
3. Type and results appear instantly

### I want to merge duplicates
1. Check boxes on 2 contacts
2. Click "Merge Selected"
3. Done! Duplicates combined

### I want to deploy to the web
1. Read: [docs/ARCHITECTURE.md](./docs/ARCHITECTURE.md)
