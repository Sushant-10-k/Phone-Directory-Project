# Contact Book Application - Quick Start

## 🚀 Get Started in 5 Minutes

### Step 1: Setup Backend

```bash
cd backend
npm install
npm start
```

Backend runs on `http://localhost:3001`

### Step 2: Setup Frontend (new terminal)

```bash
cd frontend
npm install
npm start
```

Frontend runs on `http://localhost:3000`

### Step 3: Open Browser

Go to `http://localhost:3000` and start managing contacts!

---

## 📋 Features

✅ **Add Contacts** - Enter complete contact information  
✅ **Search** - Find contacts by name, email, or phone  
✅ **Edit** - Update contact information anytime  
✅ **Delete** - Remove contacts with confirmation  
✅ **Merge** - Combine duplicate contacts intelligently
